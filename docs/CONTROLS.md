## Menu Options

### Main Menu
1. **Toggle Aimbot** - Enable/disable aim assistance
2. **Trigger Bot** - Auto-shoot when target locked
3. **Show Visualization** - Show/hide detection boxes
4. **Smooth Factor** - Aim smoothness (0.01-1.0)
5. **Noise Intensity** - Human-like jitter (0.0-5.0)
6. **Max Offset** - Max movement per frame (10-500)
7. **Speed Multiplier** - Overall speed (0.1-5.0)
8. **Aim Height** - Aim point (2=head, 100=feet)
9. **FOV Size** - Detection area (100-1000)
10. **Confidence** - Detection threshold (0.1-0.9)
11. **Humanization** - Enable/disable random noise
12. **Advanced Settings** - Additional options
13. **Save Configuration** - Save current settings
14. **Reset to Default** - Restore default values
15. **Exit Program** - Close application

### Advanced Menu
1. **Mouse Delay** - Input delay (0.0001-0.01)
2. **IOU Threshold** - Detection overlap (0.1-0.9)
3. **Targeting Scale** - Sensitivity multiplier (0.1-3.0)
4. **Back to Main** - Return to main menu

## Game Integration

### Supported Games
- First-person shooters
- Third-person games
- Any game with visible player models

### Setup Instructions
1. Run game in **Windowed** or **Borderless Windowed** mode
2. Launch ADS Aimbot
3. Adjust FOV to match game's view
4. Tweak confidence for your game
5. Hold RMB to activate aiming

### Best Practices
- Start with low sensitivity
- Gradually increase smoothness
- Add humanization for natural movement
- Test in training mode first
- Adjust for each game individually

## Troubleshooting Controls

### Menu Not Opening
- Ensure INSERT key works
- Check for key conflicts
- Restart application

### Controls Not Working
- Run as Administrator
- Check game compatibility
- Verify right mouse button detection

### Sensitivity Issues
- Adjust Targeting Scale
- Modify Smooth Factor
- Change Max Offset
